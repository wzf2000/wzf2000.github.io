题意：

  • 给出两个字符串$p$和$s$和一个正整数$k$，要求构造长度为$k$的$01$串，满足以下要求：

  • 输出满足要求的字典序最小的$01$串。
  • $|p| \leq 10^6,|s| \leq 200,k \leq 2000$
