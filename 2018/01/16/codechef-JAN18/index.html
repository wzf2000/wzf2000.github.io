A humongous Query

题意：

  • 给一个$01$串，$q$次询问与它差异最小（不同位数最少）的满足形如$1010\cdots 10$的子序列有$x$个的串与给定串的最小差异。
  • $n \leq 32,q \leq 10,x \leq 10^6$

题解：

  • 首先考虑拆成两半，枚举算出每种$01$串形如$1010\cdots10$等四种子序列的个数。
  • 设左边$1$开头$0$结尾的有$a$个，$1$开头$1$结尾的有$b$个。
  • 设右边$1$开头$0$结尾的有$c$个，$0$开头$0$结尾的有$d$个。
  • 那么输入的$x=(a+1) \times (c+1)-1+b \times d$
  • 那么$a,b,c,d$必然至少有一个大于等于$\sqrt{\frac{x}{2}}$的。
  • 考虑枚举其中一个串，然后只在这个串的某个值满足的情况下枚举另一个串的可能的两个值。通过数组获取最小值。
  • 复杂度$O(15\times 2^{15}+q\times 2^{\frac{n}{2}}\times \sqrt{x})$

Killjee and k-th letter

题意：

  • 给定一个长度为$n$的字符串，$Q$询问将他的所有子串按字典序从小到大排列后得到的串第$x$位是多少。（强制在线）
  • $n,Q\leq 2\times 10^5$

题解：

  • 首先可以建出原串的后缀树，发现字典序从小到大，相当于后缀树的某种$dfs$序。
  • 然后就可以通过在$dfs$序上二分，和在边上二分解决。
  • 后缀树可以通过倒着建立后缀自动机建出。

Simplify the Square Root

题意：

  • 询问第$n$个有平方因子的数。
  • $n \leq 10^{18}$

题解：

  • 首先第$n$个问题太难，我们考虑另一个问题：
    • 前$n$个正整数中有多少个有平方因子。
  • 我们可以一波容斥发现，即求：
  • 考虑暴力计算前一部分的值，同一部分按照$\lfloor \frac{n}{i^2} \rfloor$分类，用杜教筛计算莫比乌斯函数的前缀和。
  • 然而直接二分还是会$TLE$，所以我们考虑迭代。
  • 有一个不知道从哪里来的定理告诉我们$\mu(i)=1$和$\mu(i)=-1$的概率均为$\frac{3}{\pi^2}$，所以我们可以估算出所求答案在$n\times \frac{\pi^2-6}{\pi^2}$附近，然后在差的个数小于等于$1000$时，暴力往前或往后跑，通过$O(n^{\frac{1}{3}})$的算法判断其是否含平方因子。
  • 复杂度玄学，似乎有人说$O(n^{\frac{2}{5}})$，有人说$O(n^{\frac{1}{3}})$？？
