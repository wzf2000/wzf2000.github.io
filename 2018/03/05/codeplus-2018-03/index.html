白金元首与莫斯科

题意：

  • 给你一个$n\times m$的网格图，有一些点有障碍，可以像多米诺骨牌一样覆盖，但一个点只能覆盖一次且不需要覆盖满，询问每个点变成障碍点后覆盖的方案数。
  • $n,m\le 17$

题解：

  • 显然这道题是一道轮廓线$\rm dp$（好）题。
  • 如果暴力枚举障碍点，复杂度会是$O(n^2m^22^m)$。
  • 显然无法接受。
  • 我们考虑从头尾$\rm dp$，然后合并。
  • 显然两边的插头应该一模一样，而且当前这个点不应该有任何插头。
  • 所以这样时间复杂度就是$O(nm2^m)$。

博弈论与概率统计

题意：

  • 两个人玩游戏，进行了$n+m$轮，第一个人赢的概率是$p$，他赢了$n$轮。
  • 每次第一个人赢了，得分$+1$，否则$-1$，但分数不会低于$0$。
  • $1\le T,n,m\le 2.5\times 10^5$。

题解：

  • 显然$p$并没有用。
  • 我们发现那个分数不小于$0$非常奇怪。
  • 所以我们可以用这种方式来计算分数：
    • 设最后一次为$0$的状态为$(x,y)$。（赢了$x$轮，输了$y$轮）
    • 那么分数为$(n-x)-(m-y)$。
  • 然后用平面直角坐标系中的坐标表示，我们发现就是经过了$y=x+k$的直线的路径分数至少为$k+n-m$。
  • 最大分数显然是$n$。
  • 如果$n\le m$，显然最小分数是$0$。
  • 否则最小分数是$n-m$。
  • 考虑经过了$y=x+k$的路径方案数。
  • 我们可以将$(0,0)$关于$y=x+k$对称。
  • 得到$(-k,k)$与$(n,m)$的路径数$C_{n+m}^{n+k}$就是需要累加的方案数。
  • 然后除去一些特殊情况，也就是需要求出$\sum_{i=0}^{lim}C_{n+m}^{i}$。
  • 如果暴力就是$O(Tn)$的。
  • 配合后面一些部分分也只有$50$分。
  • 但是可以分块预处理出所有$pos=k\times \sqrt{n}$的$S(now,pos)=\sum_{i=0}^{pos}C_{i}^{now}$的值。
  • 然后只需要可以$O(1)$从$S(i,j)$转移到$S(i+1,j)$即可。
  • 然后我们就可以$O((T+n)\sqrt{n})$的复杂度解决这道题。
